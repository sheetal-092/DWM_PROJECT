# 💰 Money Laundering Detection Using Big Data and Machine Learning

## 📌 Overview
This project focuses on detecting suspicious financial transactions that may indicate money laundering activities. It leverages Big Data technologies like Apache Spark along with Machine Learning models to analyze large-scale transaction data and classify transactions as normal or suspicious.

---

## 🎯 Objectives
- Analyze large-scale financial transaction data  
- Identify suspicious transaction patterns  
- Perform feature engineering for better prediction  
- Build Machine Learning models for classification  
- Handle imbalanced datasets  
- Develop a scalable fraud detection system  

---
